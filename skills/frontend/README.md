# Frontend skills

Conventions and workflows for React + TypeScript frontend development.

| Skill | Use when |
| --- | --- |
| [`react-core`](react-core/SKILL.md) | Setting up a new project, adding a new core concern, or modifying app-level infrastructure (Axios, QueryClient, router, i18n, theme, contexts, storage, Zustand layout store). |
| [`react-standard`](react-standard/SKILL.md) | Writing any new component, hook, service, type, or utility, or reviewing existing code for naming, structure, i18n, and error-handling consistency. |
| [`feature-architecture`](feature-architecture/SKILL.md) | Creating a new feature, adding a major section, or organizing a feature's files (components, hooks, services, types, schemas, constants, utils) and connecting it to routes. |
| [`tanstack-router`](tanstack-router/SKILL.md) | Creating route files, adding search params validation, route guards, loaders, or navigating programmatically with TanStack Router. |
| [`tanstack-query`](tanstack-query/SKILL.md) | Creating hooks for API calls — list/single-item queries, create/update/delete mutations, or cache invalidation — with TanStack Query v5. |
| [`tanstack-form`](tanstack-form/SKILL.md) | Creating or modifying any form (create/update/filter/modal) with TanStack Form v1's `useAppForm` hook and `App*` field components. |
| [`table-mrt`](table-mrt/SKILL.md) | Creating a list/table view for a feature using the `DataTable` (Material React Table) wrapper — columns, pagination, row actions, selection, toolbar. |
| [`mui-theme`](mui-theme/SKILL.md) | Working with colors, spacing, typography, component styling, theme overrides, or shared UI building blocks like `SectionCard`/`InfoRow`. |
